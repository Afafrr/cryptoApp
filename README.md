## General info
Project uses Coingecko API to fetch cryptocurrencies data. User can add coins to favourites by clicking a star and check his coins in favourites page.
	
## Technologies
Project is created with:
* React 18.2
* Typescript
* Tailwind
* Coingecko API
* React-dom 18.2
* React-router-dom 6.14.1
